Get a parser for a specific MIME type

Returns a parser appropriate for the requested MIME type. If no MIME type is specified or no matching format is found, returns a special parser that attempts to auto-detect the format from the content.

This method will never return null - if no suitable parser is found, it returns a parser that will attempt detection during the parse operation.

@param mimeType Optional MIME type to specify the format to parse @return A parser for the requested format, or an auto-detecting parser

Implementation

RdfParser getParser(String? mimeType) {
  final format = getFormat(mimeType);
  if (format != null) {
    return format.createParser();
  }

  // No specific format found, return a detecting parser
  return FormatDetectingParser(this);
}
