Creates a specialized RdfParser for a specific format.

The factory selects the appropriate parser based on the content type. If no content type is provided, it should return a parser that can auto-detect the format from the input.

Parameters:

  • contentType MIME type that determines which parser to create (e.g., "text/turtle"). If null, returns an auto-detecting parser.

Returns:

  • An RdfParser instance for the specified content type.

Throws:

  • FormatNotSupportedException if the requested format is not supported.
