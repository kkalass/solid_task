JsonLdParser

JsonLdParser class

A parser for JSON-LD (JSON for Linked Data) format.

JSON-LD is a lightweight Linked Data format based on JSON. It provides a way to help JSON data interoperate at Web-scale by adding semantic context to JSON data. This parser supports:

  • Basic JSON-LD document parsing
  • Subject-predicate-object triple extraction
  • Context resolution for compact IRIs
  • Graph structure parsing (@graph)
  • Type coercion

Example usage:

final parser = JsonLdParser('''
  {
    "@context": {
      "name": "http://xmlns.com/foaf/0.1/name"
    },
    "@id": "http://example.com/me",
    "name": "John Doe"
  }
''', baseUri: 'http://example.com/');
final triples = parser.parse();
