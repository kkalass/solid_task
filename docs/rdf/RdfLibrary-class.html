RdfLibrary

RdfLibrary class final

Central facade for the RDF library, providing access to parsing and serialization.

This class serves as the primary entry point for the RDF library, offering a simplified interface for common RDF operations. It encapsulates the complexity of parser and serializer factories, format registries, and plugin management behind a clean, user-friendly API.

The class follows IoC principles by accepting dependencies in its constructor, making it suitable for dependency injection and improving testability.

For most use cases, the RdfLibrary.withStandardFormats factory constructor provides a pre-configured instance with standard formats registered.
